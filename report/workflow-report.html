6 Reflection

The biggest surprise was that the "fresh clone" was broken before any feature work began: the starter left pydantic and bcrypt unpinned, so pip pulled versions too new for SQLModel 0.0.22 and passlib 1.7.4, and the app wouldn't even boot. Pinning both (2.9.2 / 4.3.0) and matching Python 3.12 fixed it — and that quirk became the most valuable line in the project CLAUDE.md, exactly the kind a fresh model could never guess. The TDD loop paid off most on the mention parser, where forcing an email-address edge case into the failing test first meant the naive @(\w+) regex never survived. If I did it again I'd use git add -A rather than git commit -am from the start, since -am silently skipped new files and briefly left implementation code out of its intended feat: commit.
